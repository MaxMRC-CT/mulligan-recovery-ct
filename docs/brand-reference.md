# Mulligan Brand Reference (v2)

Source of truth: `/Users/maxbauscher/Downloads/Mulligan_Recovery_Centers_Brand_Kit_v2.zip`

## Core Direction
- Minimal, calm, modern
- Black and white base with restrained orange accents
- Rounded corners, soft shadows, no gradients
- Reduce cognitive load and make next steps obvious

## Web Content Rules
- Short paragraphs (2-3 lines)
- 8th-10th grade reading level
- Explain terms once, then plain language
- CTA language prefers: "Talk to someone today", "See if Mulligan is right for you"

## UI Rules
- White space over decoration
- Mobile-first readability
- Stats carousel should support credibility without hype
- Calm transitions only

## Asset Mapping
- Full logo: `public/logo.png`
- Circle logo: `public/logo-circle.png`

## Current Token Strategy
- Base: neutral black/white
- Accent: restrained orange for CTA and highlights
- No gradient backgrounds

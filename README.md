# Mulligan Recovery Centers of CT Website

Production-ready marketing website scaffold built with Next.js App Router, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

## Build and deploy

```bash
npm run build
npm run start
```

Deploy directly to Vercel. The site uses static and server-rendered App Router pages plus markdown-based resource posts.

## Content locations

- Pages: `app/**/page.tsx`
- Shared components: `components/*.tsx`
- Resource posts: `content/resources/*.md`
- Blog parsing utility: `lib/blog.ts`

## Compliance note

This scaffold includes HIPAA-aware messaging and legal placeholders, but legal/compliance teams should review all public-facing language before production launch.
